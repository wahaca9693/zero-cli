# ZERO CLI Companion

The ZERO CLI Companion extension pairs with
[ZERO CLI](https://github.com/google-zero/zero-cli). This extension is
compatible with both VS Code and VS Code forks.

# Features

- Open Editor File Context: ZERO CLI gains awareness of the files you have
  open in your editor, providing it with a richer understanding of your
  project's structure and content.

- Selection Context: ZERO CLI can easily access your cursor's position and
  selected text within the editor, giving it valuable context directly from your
  current work.

- Native Diffing: Seamlessly view, modify, and accept code changes suggested by
  ZERO CLI directly within the editor.

- Launch ZERO CLI: Quickly start a new ZERO CLI session from the Command
  Palette (Cmd+Shift+P or Ctrl+Shift+P) by running the "ZERO CLI: Run"
  command.

# Requirements

To use this extension, you'll need:

- VS Code version 1.99.0 or newer
- ZERO CLI (installed separately) running within the integrated terminal
