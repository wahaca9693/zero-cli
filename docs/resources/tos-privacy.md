## Supported authentication methods

Your authentication method refers to the method you use to log into and access
Google’s services with ZERO CLI. Supported authentication methods include:

- Logging in with your Google account to ZERO Code Assist.
- Using an API key with ZERO Developer API.
- Using an API key with Vertex AI GenAI API.

The Terms of Service and Privacy Notices applicable to the aforementioned Google
services are set forth in the table below.

If you log in with your Google account and you do not already have a ZERO Code
Assist account associated with your Google account, you will be directed to the
sign up flow for ZERO Code Assist for individuals. If your Google account is
managed by your organization, your administrator may not permit access to ZERO
Code Assist for individuals. See the
[ZERO Code Assist for individuals FAQs](https://developers.google.com/zero-code-assist/resources/faqs)
for further information.
