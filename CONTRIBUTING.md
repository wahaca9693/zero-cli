# كيف تساهم في المشروع

نرحب بمساهماتكم في هذا المشروع!

## عملية المساهمة

### 1. ابحث عن مشكلة
ابحث عن مشكلة على صفحة Issues في GitHub أو أنشئ واحدة جديدة.

### 2. Fork المشروع
أنشئ نسخة خاصة من المشروع على حسابك.

### 3. أنشئ فرع جديد
أنشئ فرع جديد للعمل عليه.

### 4. قدم التغييرات
بعد الانتهاء، قدم Pull Request للمراجعة.

### 5. المراجعة
سنراجع تغييراتك وندمجها في المشروع.

## نصائح

- اتبع معايير الكود الموجودة في المشروع
- اكتب اختبارات عند الإمكان
- وثّق التغييرات بشكل جيد
